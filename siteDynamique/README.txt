Bienvenue sur notre partie site dynamique, je vais vous expliquer comment faire fonctionner le site avec la bdd sur votre ordinateur !

-Placer le dossier siteDynamique dans votre dossier C:/wamp64/www
	
-Lancer votre serveur wamp

-Dans votre navigateur taper : http://localhost/phpmyadmin/index.php pour accèder a votre base de données

-Utilisateur par défault = root ; pas de mot de passe

-Cliquez sur le menu à gauche : Nouvelle base de données

-Nom = abi ; langage = utf8_general_ci ; Créer la bdd

-Allez sur celle-ci et cliquez sur le menu en haut Importer

-Ajouter le fichier abi.sql se trouvant dans le dossier siteDynamique et exécuter l'importation pour avoir accès à notre bdd sur votre pc

-Dans votre navigateur taper : localhost/siteDynamique/index.php

-Pour tester l'intranet avec un compte administrateur : Login = 1 ; Mot de passe = admin

-Pour tester l'intranet avec un compte utilisateur : Login =  ; Mot de passe = admin

Bonne navigation !
